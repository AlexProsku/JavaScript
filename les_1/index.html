4. *Самостоятельно разобраться с атрибутами тега script (async и defer)
Ответ:Б
Атрибут defer в script используется для того, чтобы не блокировать дальнейшую загрузку/обработку html, скрипт выполнится по готовности (когда загрузится), при этом последовательность выполнения скриптов сохраняется
Атрибут async отличается от defer тем, что последовательность выполнения скриптов не соблюдается и подобные скрипты выполняются по мере загрузки.
