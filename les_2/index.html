Практическое задание к Уроку 2

1. Дан код:
var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2 Оператор инкремента - вначале происходит прибавление 1, а затем происходит вывод alert
d = b++; alert(d); // 1 Тоже оператор инкремента, но он выполняется уже после вызова переменной d
c = (2+ ++a); alert(c); // 5 Происходит операция (2+1+1+1) и вывод alert
d = (2+ b++); alert(d); // 4 Происходи операция (2+1+1 потом появление переменной и инкремент 1) и после операции скобок вызывается alert
alert(a); // 3 Изначально a=1, но потом с a два раза делали инкремент
alert(b); // 3 Изначально b=1, но потом с b два раза делали инкремент
Почему код даёт именно такие результаты?
Ответы: Указаны в тексте задания

2. Чему будет равен x в примере ниже? var a = 2; var x = 1 + (a *= 2);
Ответ: x = 5

3. Объявить две целочисленные переменные a и b и задать им произвольные начальные значения. Затем написать скрипт, который работает по следующему принципу: если a и b положительные, вывести их разность; если а и b отрицательные, вывести их произведение; если а и b разных знаков, вывести их сумму; ноль можно считать положительным числом.
Решение: в файле скрипта и выводах alert

4. Присвоить переменной а значение в промежутке [0..15]. С помощью оператора switch организовать вывод чисел от a до 15.
Решение: в файле скрипта

5. Реализовать основные 4 арифметические операции в виде функций с двумя параметрами. Обязательно использовать оператор return.
Решение: в файле скрипта

6. Реализовать функцию с тремя параметрами: function mathOperation(arg1, arg2, operation), где arg1, arg2 – значения аргументов, operation – строка с названием операции. В зависимости от переданного значения операции выполнить одну из арифметических операций (использовать функции из пункта 3) и вернуть полученное значение (использовать switch).

7. *с. Попробуйте объяснить результат.
Ответ: При сравнении 0 и null результат следующий:
null > 0; // false
null == 0; // false
null >= 0; // true
null <= 0; // true
Это происходит из-за попытки приведения к числу при сравнении
